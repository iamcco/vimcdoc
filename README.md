# VIM中文文档(Vim Chinese Documentation) #

## 使用方法 ##

### Vundle安装 ###

在`vim的配置文件_vimrc`文件中加入一句

    Plugin 'iamcco/vimcdoc'

重启VIM或:source $VIM/_vimrc重新加载vim的配置文件，运行`:PluginInstall`即可完成安装
